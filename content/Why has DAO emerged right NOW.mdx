# Why has DAO emerged right NOW

还不了解 DAO 是什么的朋友，可以看一看[这篇文章](https://mp.weixin.qq.com/s/m4TPEKutkXUCTLmo7vuC2w)）。简单来说，DAO, **Decentraolized autonomous organization**, 就是让**分散的**参与者，为了某个**使命**，在某个**治理机制**下，在**某条区块链上**，自发地进行**协调、组织**，并**完成任务**。

在我有对 dWeb 有印象以来，DAO 在中文语境里被我所了解到的时间是今年的1月左右。刚开始，我对这个概念并不是特别感冒：不过是一种对于线上协作社区的，更 fancy 一些的叫法罢了。 github 上进行 collaborate 的方式不就是类似于这种 DAO 吗？

事实证明我错了。5月份的时候 DAO 在我的视界内再次活跃了起来，并且这次是带着项目来的了。当时因为手头事情比较多，也还是没有具体了解。到了前几个星期，终于跳进了这个兔子洞，去 see for myself.

然后我被惊到了。这种震惊不只是理性上的，震惊于 DAO 社区的规整程度，更是被社区内大家愿意贡献的热情给感染到了。引用一下 [Gridea](https://gridea.dev/) 首页的句式：“可以，协作那个劲儿又回来了”。

关于 DAO 社区的细节，这篇文章暂时不做讨论，在参与协作的过程中，有一个问题一直作为后台程序运行着：为什么 DAO 现在呈现出来了？之前 github 上确实有很多很好的协作项目，为什么没有变得像 DAO 这样的 viral，吸引了那么多各界人士（指 Bankless DAO 中不止有 coder，也有法务专家，设计师等）？
